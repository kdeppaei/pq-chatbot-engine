# pq-chatbot-engine
A no-API hybrid chatbot engine built with PQ logic, rule-based reasoning, TF-IDF retrieval, and explainable intent classification.

## Live Demo

GitHub Pages URL:

https://kdeppaei.github.io/pq-chatbot-engine/

## What Runs In The Browser

- Rule-based P -> Q inference
- FAQ retrieval with tokenization, inverted index, TF-IDF, and cosine similarity
- From-scratch Naive Bayes intent classifier
- Explainable debug panel showing facts, matched rules, retrieval scores, intent probabilities, and final decision reason

No external AI API is used.
